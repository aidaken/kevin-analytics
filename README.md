## Что делаем
VIP даёт ~28% выручки — это важно, но не единственный источник денег.
Выручка размазана по всем группам, поэтому стратегия “только VIP” будет узкой.

**Решение:**
- VIP: удержание (не потерять этих ребят)
- Core: апгрейд в VIP (самый понятный рост)
- Остальные: лёгкие “реактивации”, без фанатизма

**Почему:**
выручка не концентрируется в одной группе — значит выигрывает стратегия “удержать + подрастить”, а не “VIP-only”.

**Ожидаемый эффект:**
более стабильная выручка + рост за счёт перевода части Core в VIP.
